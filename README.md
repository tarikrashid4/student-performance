# student-performance
student performance
The coding is in matlab; there are six models in six differnt folders:

1-Recurrent NN-GWO
2-Recurrent NN-Modified GWO
3-Cascadeing NN-GWO
4-Cascading NN-Modified GWO
5-MLP-GWO
6-MLP- Modified GWO

To run any model, select its folder and run the My_Main.m file.
